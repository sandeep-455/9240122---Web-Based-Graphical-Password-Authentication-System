Graphical Password Authentication System

Protect accounts from brute-force and keylogging attacks. Set your password by selecting a pattern of secret points on a custom image instead of typing plain text.

Get Started Now Access Account

Why Graphical Passwords?

Traditional text-based passwords are hard to remember and easily stolen by keyloggers. Graphical passwords use images, offering high recall rate and robust security.

Keylogger Resistant

Since you click on coordinates instead of typing letters, keyboard-recording spywares cannot record your credentials.

High Memorability

Human brains remember images and spatial locations much better than complex alpha-numeric string patterns.

Anti-Brute Force

Includes coordinate tolerance protection, strict 30-second lockouts, and email alerts when unauthorized attempts occur.

How the System Works

A simple four-step authentication pipeline designed for maximum speed and absolute security.

1

Sign Up

Enter a unique username and your active email address to receive validation codes.

2

Select Image

Choose from our default high-detail pictures or upload your own personalized image.

3

Set Click Sequence

Click between 3 to 6 points on the image in a specific order to set your graphical password.

4

Verify & Two-Factor OTP

Log in by clicking the correct sequence. Complete verification using a 6-digit email OTP.

System Architecture

This college-level capstone project is built on clean, modern, and light-weight backend and frontend systems, making it easily deployable and testable.

Flask & Python

App routing, secure session management, coordinate matching verification, and SMTP-based alert controls are handled by Python backend.

SQLite 3 Database

A simple database containing relational schema for usernames, email addresses, image path URLs, and JSON-encoded target coordinates.

Flask Framework Python 3 SQLite 3 HTML5 / CSS3 Vanilla JavaScript SMTP Email API

Coordinate Tolerance

Our algorithm permits a 3px click offset to accommodate touch screens and normal finger tremors while keeping validation precise.

Automatic Locking System

After 3 wrong pattern attempts, the login is temporarily frozen. The system instantly issues an alert email to the registered account.
